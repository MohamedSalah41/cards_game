# Assets Guide

## Images  (`assets/images/`)

| File              | Purpose                                  |
|-------------------|------------------------------------------|
| `bg.jpg`          | Full-page background image               |
| `card-back.png`   | Back face shown on every card (shared)   |
| `card-1.png`      | Front face — pair 1                      |
| `card-2.png`      | Front face — pair 2                      |
| `card-3.png`      | Front face — pair 3                      |
| `card-4.png`      | Front face — pair 4                      |
| `card-5.png`      | Front face — pair 5                      |
| `card-6.png`      | Front face — pair 6                      |
| `card-7.png`      | Front face — pair 7                      |
| `card-8.png`      | Front face — pair 8                      |
| `card-9.png`      | Front face — pair 9                      |
| `card-10.png`     | Front face — pair 10                     |

Tip: any square images work well (e.g. 200×300 px PNG).

## Audio  (`assets/audio/`)

| File              | Triggered when …                          |
|-------------------|-------------------------------------------|
| `background.mp3`  | Game starts (loops)                       |
| `flip.mp3`        | A card is clicked and flipped             |
| `match.mp3`       | Two flipped cards share the same pair     |
| `mismatch.mp3`    | Two flipped cards do NOT match            |

Free sources: [freesound.org](https://freesound.org), [pixabay.com/music](https://pixabay.com/music/)
